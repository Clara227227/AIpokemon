# Pokémon Champions Checker & Simulator

対戦特化ゲーム『Pokémon Champions』を対象とした、手持ち6匹の相性チェッカー、仮想敵シミュレーター、およびポケモンデータ管理システムです。

## 主な機能
1. **チーム診断 (相性チェッカー)**
   - 6匹の手持ちポケモンの防御タイプ一貫性（弱点タイプ）をヒートマップ表示。
   - チーム全員の技範囲から攻撃一貫性（抜群を取れるタイプ）を評価。
   - チーム全体のタイプバランスの穴や打点不足を自動評価し、テキストでアドバイス。
2. **仮想敵シミュレーター**
   - 現環境（レギュレーションM-B）の主要な仮想敵（対策必須のポケモン）に対する勝率・有利不利を自動シミュレート。
   - チーム内で最も重い（勝率が低い）敵を特定し、補完するための具体的な入替候補と理由を自動提案。
3. **ポケモンDB管理**
   - 登録されている全ポケモンの編集（タイプ、種族値、覚える技、メガシンカ）が可能。
   - PokeAPIから英語名や図鑑番号を用いて、最新データを自動取得して登録可能。
   - 編集したデータベースのJSONエクスポート・インポート機能を搭載。

## 技術スタック
- フロントエンド: Pure HTML / Vanilla CSS / Modern JavaScript (ES Modules)
- データベース: LocalStorage
- 外部API: PokeAPI
- デプロイ環境: Vercel (静的ホスティング)
